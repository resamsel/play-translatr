Integrated solution designed for ease of use

Translators

Translate a project to a new language, check the outcome in the preview pane, see how other languages translate that same key. This is all possible on the same page.

Developers

Adding new keys is as easy as a few clicks. Translate them to your default language, pull them into your locale development environment, then check the result.

Operators

Use the power of the API to integrate translations into your applications. The Translatr CLI allows you to pull, push, or create new languages and keys. You could also use the API directly to integrate your own workflow.

Activity

The activity graph shows the usage of Translatr. The change history contains contributions to users, projects, locales, keys, and translations. That way the activity of a project can be estimated, as well as the activity of individual users.

Single Source of Truth

Use Translatr as source of internationalization - translator and developers can both add their content to the same platform.

Translatr CLI

Your pipeline can then read the current state and put it into your applications. It's never been so easy - use the Translatr CLI to assist you with that task.

Internationalization as Code

Keep translations in your repository, as part of your code base. Use it as your single source of truth, including version control, access control and change history.
